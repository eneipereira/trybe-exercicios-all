Soco a 80km/h: Conheça o Stomatopoda

Fatos sobre o Stomatopoda

Informações Gerais

O stomatopoda, conhecido fora do Brasil como "mantis shrimp", também conhecido como estomatópode, tamarutaca, lacraia-do-mar, lagosta boxeadora. É um crustáceo carnívoro, cujo nome científico é Odontodactyllus scyllarus

Apresenta classificação científica conforme tabela abaixo:

Reino Filo Subfilo Classe Subclasse Ordem
